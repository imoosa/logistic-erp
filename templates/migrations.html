Platform Database Changes

⚠️ Manual SQL Required

The platform database (logistic_erp) contains system tables like:

  • subscription_plans - Subscription plans configuration
  • registered_users - User accounts (super_admin, owners)
  • companies - Company information
  • backup_records - Backup history
  • backup_schedules - Scheduled backup settings

To alter the platform database, run SQL directly using:

mysql -u root -p logistic_erp

-- Example: Add a column
ALTER TABLE companies ADD COLUMN tax_registration_number VARCHAR(50) NULL;

-- Example: Create a new table
CREATE TABLE IF NOT EXISTS system_settings (
    id INT AUTO_INCREMENT PRIMARY KEY,
    setting_key VARCHAR(100) NOT NULL,
    setting_value TEXT,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

Note: Platform changes affect all companies and require careful testing.
